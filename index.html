Game Overview

BeeZii is a fast-paced endless flying game where players control a bee, dodging obstacles and collecting honey drops to achieve the highest score.

Gameplay Mechanics

Controls

  • Tap to make the bee jump.
  • Gravity pulls the bee down when idle.

Obstacles

  • Birds move in straight or wave patterns.
  • Carnivorous plants trigger an attack animation when close.
  • Honey drops spawn at variable rates depending on difficulty.

Development Guidelines

Project Structure

  • Scripts/Controllers: Handles movement and game logic.
  • Scripts/Managers: Handles UI, Ads, Audio, and Scores.
  • Prefabs/: Stores reusable game objects like Bee, Honey, and Obstacles.
  • Scenes/: Main Menu, Game, Game Over.

Technical Stack

  • Engine: Unity (C#)
  • UI Framework: Unity UI (Canvas-based)
  • Audio: Unity Audio Source
  • Ad Integration: Unity Ads
  • Sharing: Platform-dependent (Android Intent / iOS UIActivityViewController)

Game Difficulty Progression

The game increases in difficulty over time by:

  • Reducing the time between obstacle spawns.
  • Introducing birds that follow wave-like movements.
  • Randomizing honey drop values and rarity.
